# delete-header-in-CSVfile

This scripts uses the os aand csv mmodules in python.
it checks if a file in the current directory is a .csv file

for each .csv file, it skips over the first line and then writes the remaining part of the file.
the nerw file without the header is written to a new folder called "Headers removed" 
within the same directory.

this program is run on the command line
